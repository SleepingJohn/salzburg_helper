# Salzburg Citizen Report

A React Native Expo prototype for a multilingual citizen reporting app.

## Features

- Welcome landing page with Stadt Salzburg branding
- Voice recording flow with a simple waveform
- Mock AI processing screen for transcription, translation, and routing
- Success confirmation in multiple languages
- Issue tracking timeline

## Run locally

1. Install dependencies

```bash
npm install
```

2. Start Expo

```bash
npm start
```

3. Open in the Expo Go app or simulator
